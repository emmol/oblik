# Проект сайта для магазина Oblik
### Описание проекта
Учебный проект - для магазина обуви
### Технологии
Python 3.10
Django 4.1.5
### Инструкции по запуску проекта
```
pip install -r requirements.txt
```
- В папке с файлом manage.py выполните команду:
```
python3 manage.py runserver
```
